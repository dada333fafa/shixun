# 🚀 Vue前端启动指南

## 前置条件

1. ✅ Node.js 已安装（建议 v16+）
2. ✅ 后端服务正在运行（http://localhost:5001）

## 启动步骤

### 第一步：安装依赖

```bash
cd d:\vallege_studetn\vallege_student_help_qian
npm install
```

### 第二步：启动Vue开发服务器

```bash
npm run dev
```

### 第三步：访问应用

打开浏览器访问：**http://localhost:8080**

## 🔧 技术特点

### Vue 3 Composition API
使用 `<script setup>` 语法，更简洁的代码组织

### Vue Router
单页面应用，无刷新路由跳转

### Vite代理
自动将 `/api` 请求代理到后端 `http://localhost:5001`

### 响应式设计
保持原有的所有样式和布局

## 💡 开发提示

1. **热更新**：修改代码后自动刷新，无需手动刷新浏览器
2. **开发者工具**：安装 Vue DevTools 浏览器扩展进行调试
3. **控制台日志**：按 F12 查看Console输出
4. **API调用**：所有API函数在 `src/utils/api.js` 中

## ⚠️ 注意事项

1. **不要修改后端**：只操作 `vallege_student_help_qian` 文件夹
2. **保持后端运行**：Vue项目需要后端API才能正常工作
3. **端口占用**：如果8080端口被占用，修改 `vite.config.js` 中的端口配置

## 🆘 常见问题

### Q: 启动时报错 "port 8080 already in use"
A: 关闭占用8080端口的程序，或修改 vite.config.js 中的端口

### Q: API请求失败
A: 确保后端服务在 http://localhost:5001 正常运行

### Q: 页面空白
A: 检查浏览器控制台错误信息，确保npm install已完成
