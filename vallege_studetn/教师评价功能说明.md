# 教师评价功能说明

## 功能概述

本功能实现了教师对学生打分和评语的功能，家长可以在学习报告中查看教师的评价信息。

## 功能特性

### 1. 教师端功能
- ✅ 在学生管理界面为辅导的学生打分（0-100分）
- ✅ 为学生编写详细评语
- ✅ 选择评价科目（综合、数学、语文、英语、物理、化学）
- ✅ 支持更新已有评价
- ✅ 自动验证教师是否有权限评价该学生

### 2. 家长端功能
- ✅ 在学习报告页面查看所有教师对孩子的评价
- ✅ 显示教师姓名、评价科目、分数、评语和评价时间
- ✅ 美观的卡片式展示
- ✅ 支持多个教师的评价展示

### 3. 数据存储
- ✅ 评价数据存储在MongoDB数据库中
- ✅ 包含学生ID、教师ID、分数、评语、科目、评价时间等字段
- ✅ 支持同一教师对同一学生在不同科目的多次评价

## 技术实现

### 后端实现

#### 1. 数据模型 (TeacherEvaluation.js)
```javascript
{
  student: ObjectId,      // 学生ID
  teacher: ObjectId,      // 教师ID
  score: Number,          // 分数 (0-100)
  comment: String,        // 评语
  subject: String,        // 科目 (默认: 综合)
  evaluationDate: Date    // 评价时间
}
```

#### 2. API接口

**添加或更新评价**
- 路径: `POST /api/v1/teacher-evaluations`
- 需要认证: 是
- 请求体:
```json
{
  "studentId": "学生ID",
  "score": 85,
  "comment": "评语内容",
  "subject": "综合"
}
```

**获取学生的所有评价（家长端）**
- 路径: `GET /api/v1/teacher-evaluations/student/:studentId`
- 需要认证: 是
- 返回: 该学生的所有评价列表

**获取教师对某学生的评价**
- 路径: `GET /api/v1/teacher-evaluations/student/:studentId/my-evaluation`
- 需要认证: 是
- 返回: 当前教师对该学生的最新评价

### 前端实现

#### 1. 教师端 (StudentManagement.vue)
- 在学生列表中添加"评价"按钮
- 点击按钮弹出评价对话框
- 表单包含：分数输入、评语文本框、科目选择
- 自动加载已有评价（如果存在）
- 提交时进行数据验证

#### 2. 家长端 (ParentLearningReport.vue)
- 在学习报告页面添加"教师评价"区域
- 以卡片形式展示每个教师的评价
- 显示教师姓名、科目、分数、评语和评价时间
- 如果没有评价则显示提示信息

## 使用方法

### 教师端使用步骤

1. 登录教师账号
2. 进入"学生管理"页面
3. 找到要评价的学生（状态必须为"活跃"）
4. 点击"评价"按钮
5. 在弹出的对话框中填写：
   - 分数（0-100）
   - 评语（详细描述学生学习情况）
   - 科目（可选，默认为"综合"）
6. 点击"提交评价"按钮
7. 系统会提示保存成功

### 家长端查看步骤

1. 登录家长账号
2. 进入"学习报告"页面
3. 选择要查看的孩子
4. 滚动到页面底部的"教师评价"区域
5. 查看所有教师对该孩子的评价
   - 教师姓名
   - 评价科目
   - 分数
   - 评语内容
   - 评价时间

## 注意事项

1. **教师权限**：只有与学生建立辅导关系的教师才能评价该学生
2. **分数范围**：必须在0-100之间
3. **必填字段**：分数和评语为必填项
4. **评价更新**：同一教师对同一学生同一科目的评价会被更新
5. **数据显示**：家长端按评价时间倒序显示，最新的在前

## 故障排查

### 问题1：教师看不到"评价"按钮
- 检查学生状态是否为"活跃"
- 检查教师是否与该学生建立了辅导关系

### 问题2：提交评价失败
- 检查网络连接
- 检查Token是否有效
- 查看浏览器控制台错误信息
- 查看后端服务器日志

### 问题3：家长看不到评价
- 确认评价已成功保存
- 检查是否正确选择了孩子
- 查看浏览器控制台是否有错误
- 检查API返回数据
