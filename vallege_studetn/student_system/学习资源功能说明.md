# 学习资源功能说明

## ✅ 已实现的功能

### 1. 搜索功能
- 支持按标题或描述关键词搜索
- 实时过滤显示匹配的资源

### 2. 筛选功能
- **科目筛选**：数学、语文、英语、物理、化学、生物
- **资源类型筛选**：课件、教案、练习、视频、其他

### 3. 下载功能
- 点击下载按钮会记录分享历史（resource_shares表）
- 返回资源信息和文件路径
- 目前为模拟下载（因为没有实际文件）

## 📁 如何添加真实的学习资源文件

### 方法一：在项目中创建resources文件夹

1. 在后端项目根目录创建 `resources` 文件夹：
   ```
   d:\vallege_studetn\vallege_student_help_hou\resources\
   ```

2. 将你的学习资源文件放入该文件夹，例如：
   - math_fractions.pdf
   - chinese_writing.mp4
   - math_applications.pptx

3. 更新数据库中的 filePath 字段，指向实际文件路径

### 方法二：修改seedResources.js中的文件路径

编辑 `seedResources.js` 文件，修改每个资源的 `filePath` 和 `fileName`：

```javascript
{
  teacher: teacher1._id,
  title: '分数加减法练习',
  description: '...',
  resourceType: 'exercise',
  filePath: '/resources/你的文件名.pdf',  // 修改这里
  fileName: '你的文件名.pdf'              // 修改这里
}
```

然后重新运行：
```bash
node seedResources.js
```

### 方法三：通过API上传文件（需要实现文件上传功能）

目前系统已有 `/api/resources/upload` 接口，但需要：
1. 安装 multer 中间件处理文件上传
2. 配置存储路径
3. 前端添加文件上传表单

## 🔧 技术实现

### 后端API
- `GET /api/resources` - 获取资源列表（支持keyword, subject, resourceType参数）
- `POST /api/resources/:resourceId/download` - 下载资源（记录分享）

### 前端功能
- 动态加载资源列表
- 实时搜索和筛选
- 点击下载并记录历史
- 美观的卡片式展示

## 📊 数据库结构

**teaching_resources表**：
- teacher: 教师ID
- title: 资源标题
- description: 资源描述
- resourceType: 资源类型（courseware/lesson_plan/exercise/video/other）
- filePath: 文件路径
- fileName: 文件名

**resource_shares表**：
- resource: 资源ID
- student: 学生ID
- shared_at: 分享时间

## 💡 使用提示

访问 http://localhost:8080/student-resources.html 即可使用学习资源功能！
